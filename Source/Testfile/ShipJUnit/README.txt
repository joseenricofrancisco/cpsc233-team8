How to run this JUnit Test (in Eclipse):

1. Download the file "Demo2.zip" in https://github.com/joseenricofrancisco/team-8-battleship.
2. Download the file "ShipJUnit.zip" in the "Source" folder in https://github.com/joseenricofrancisco/team-8-battleship.
3. Unzip "Demo2.zip". Import all java files in Demo2 into a new project in Eclipse.
4. Unzip "ShipJUnit.zip". Import ShipTest.java into the new project's package folder which you have created in step 3.
5. Right-click on the project, navigate to "Properties", then "Java Build Path".
6. Click "Add Library", then "JUnit", then choose "JUnit4" from the drop-down menu. Hit Finish.
7. Run "ShipTest" as a JUnit4 Test.
